### 题1_1 【两数之和】
使用```哈希表```，n次循环，每次从哈希表查找target-nums[index0]的数据下标，如没有，则插入当前数据，然后下一次循环：复杂度O(n)

### 题2_53 【最大和的子数组】
```动态规划``` 遍历数组，对于每一个元素，找出以它结尾的最大子数组，从第一个元素递推，后一个元素的最大子数组于前一个元素最大子数组有关

### 题3_121 【数组中最大元素差 股票买入卖出问题】
